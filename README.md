# Ke Comer

### Proyecto realizado por el grupo C7-15-M

Maquetado de la pagina de KECOMER, donde se podrá:

- Registrarse/Loguearse.
- Ver recetas.
- Buscar recetas ingresando ingredientes.
- Guardar recetas favoritas.
- Perfil.
